# Verapamil Adverse Drug Reaction Analysis

## Research Question
What are the most common side effects reported for verapamil?

---

## Data Source
I used the FDA OpenFDA Drug Event API:
https://open.fda.gov/apis/drug/event/

This data includes reports of side effects from real patients.

---

## What I Did
- I used Python to get data from the FDA API  
- I filtered the data for verapamil  
- I counted the most common side effects  
- I made a bar chart to show the results  
- I printed some data in the terminal  

---

## Tools Used
- Python  
- requests  
- pandas  
- matplotlib  
- numpy  

---

## Results
The chart shows the most common side effects reported for verapamil.

---

## Conclusion
This project shows how FDA data can be used to see common side effects of a drug.
